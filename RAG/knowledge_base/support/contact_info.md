## Contact Tips

### For Faster Response

**Do:**
✓ Provide detailed information  
✓ Include relevant transaction IDs, URLs, screenshots  
✓ Use appropriate contact method for your issue  
✓ Be clear and concise  
✓ Include your account email  
✓ Specify urgency level if critical  

**Don't:**
✗ Email multiple addresses for same issue  
✗ Send vague inquiries  
✗ Mark non-urgent items as urgent  
✗ Include sensitive passwords or full card numbers  
✗ Use ALL CAPS (we're here to help!)  

### Information to Include

**For Account Issues:**
- Your full name
- Email associated with account
- Account username (if different)
- Description of problem
- When issue started

**For Payment Issues:**
- Transaction ID (from receipt)
- Date and time of transaction
- Amount
- Payment method used
- Error messages received

**For Campaign Issues:**
- Campaign URL or name
- Your role (creator/donor)
- Specific concern
- Screenshots if applicable

**For Technical Issues:**
- Browser and version
- Operating system
- Device type
- Steps to reproduce
- Error messages/screenshots
